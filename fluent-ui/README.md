FluentUI v9 form controls bound to [@react-typed-forms/core](https://github.com/doolse/react-typed-forms#readme).

## Install

```npm
npm install @react-typed-forms/fluent-ui
```

## Components

- `FInput` — `Input` bound to `Control<string | null | undefined>`
- `FTextarea` — `Textarea` bound to `Control<string | null | undefined>`
- `FCheckbox` — `Checkbox` bound to `Control<boolean | null | undefined>`
- `FSwitch` — `Switch` bound to `Control<boolean | null | undefined>`
- `FDropdown` — `Dropdown` bound to `Control<T | null | undefined>`
- `FRadioGroup` — `RadioGroup` bound to `Control<T | null | undefined>`
- `FSearch` — `SearchBox` bound to `Control<string | null | undefined>`
- `FNumberInput` — `SpinButton` bound to `Control<number | null | undefined>`
- `FDatePicker` — native `date`/`datetime-local` `Input` bound to `Control<string | null | undefined>` (ISO strings)
